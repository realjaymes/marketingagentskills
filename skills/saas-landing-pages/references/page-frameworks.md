# SaaS Landing Page Frameworks

## Homepage / Main Landing Page Structure

1. Hero Section (Headline, Subheadline, Bullets, Social Proof, CTA)
2. Problem to Solution Comparison
3. Features, Capabilities & Benefits (3-6 features)
4. Integrations
5. How It Works (3-5 steps)
6. FAQs
7. Second CTA Section
8. Founder's Note (optional)

## Product / Feature Page Structure

1. Hero (Feature headline, benefit statement, CTA)
2. Feature Overview (what it does, why it matters)
3. Sub-features (3-5 capabilities with benefits)
4. Use Cases (2-3 scenarios)
5. Integration Points
6. Social Proof (feature-specific testimonials)
7. CTA Section

## Solution / Persona Page Structure

1. Hero (Persona-specific headline, their language)
2. Pain Points (their specific challenges)
3. Solution Fit (how product solves their problems)
4. Key Benefits for This Persona (3-5)
5. Relevant Features
6. Case Study / Testimonial from similar user
7. CTA with persona-specific messaging

## Use Case Page Structure

1. Hero (Job-to-be-done headline)
2. Workflow Overview
3. Step-by-Step Process (3-5 steps)
4. Expected Results / Outcomes
5. Before vs After
6. Supporting Features
7. CTA

## Comparison Page Structure

1. Hero (You vs Competitor framing)
2. Quick Comparison Table
3. Key Differentiators (3-5)
4. Feature-by-Feature Breakdown
5. Switching Proof (migration ease, testimonials)
6. Pricing Comparison (if favorable)
7. FAQ (common switching questions)
8. CTA (switch now messaging)

## Integration Page Structure

1. Hero (Integration headline, partner logos)
2. What the Integration Does
3. Key Benefits (3-5)
4. Use Cases (2-3 scenarios)
5. How to Connect (simple steps)
6. Technical Details (optional)
7. CTA (connect now)

## Pricing Page Structure

1. Hero (Simple pricing message)
2. Pricing Tiers Table
3. Feature Comparison Matrix
4. Social Proof (customer count, testimonials)
5. FAQ (billing, switching tiers)
6. CTA (start with recommended plan)
7. Enterprise/Custom Option

## CTA Variations

| Type | Example |
|------|---------|
| Action | Sign up, Create account |
| Next step | Get started, Book a call |
| Value | Start recording, Show my heatmap |
| Objection | Schedule free call, No credit card |
